Feedstock for [looptools](https://feynarts.de/looptools/) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

LoopTools: evaluation of scalar and tensor one-loop integrals

## Architectures

State per architecture: see the platforms badge above or [anaconda.org/hep-forge/looptools](https://anaconda.org/hep-forge/looptools).


## Install

```bash
conda install -c hep-forge -c conda-forge looptools
```
